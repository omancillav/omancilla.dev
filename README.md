## ✨ Bienvenido a mi espacio de trabajo

Este repositorio contiene el código fuente de mi portafolio personal, construido desde cero para mostrar mi trayectoria, mis proyectos y las tecnologías con las que trabajo todos los días. Diseñé este sitio no solo para informar, sino para que funcione como una **extensión visual de mi identidad profesional**: limpio, extremadamente rápido y fácil de navegar.

### 🎯 El motor detrás de la magia

Mi objetivo era construir un sitio con un rendimiento excepcional y sin carga innecesaria de JavaScript, manteniendo al mismo tiempo una experiencia de desarrollo moderna.

## 🏗️ Arquitectura y Recorrido

Si quieres explorar el código, aquí tienes un resumen de cómo está estructurado para ser modular y escalable:

* 🧩 **Componentes Modulares (`src/components/`):** La UI está separada en piezas reutilizables como `<ExperienceItem />`, `<Project />`, `<StackCard />` y `<Badge />`.
* 🗃️ **Diseño Orientado a Datos (`src/lib/`):** El contenido y la lógica visual están separados. Archivos como `projects.ts` y `stack.ts` actúan como mi "base de datos" local.
* 🎨 **Ecosistema de Iconos (`src/assets/`):** Una extensa colección de componentes Astro (SVG) optimizados para asegurar que cada logo tecnológico se renderice con la máxima nitidez.

## 🏎️ Ponlo en marcha en tu equipo

¿Te dio curiosidad el código? Puedes clonarlo y ejecutarlo en tu entorno local en pocos segundos:

```bash
# 1. Clona este repositorio
git clone https://github.com/TU_USUARIO/omancilla.dev.git

# 2. Entra al directorio
cd omancilla.dev

# 3. Instala las dependencias (¡Recomiendo pnpm!)
pnpm install

# 4. Inicia el servidor de desarrollo
pnpm dev

```

> Abre `http://localhost:4321` en tu navegador para ver la magia en acción.
